# TikiTiki Agenda

## The best agenda... for the TikiTiki island.

Installation:

To install the program, please do:

    git clone https://github.com/manusandoval05/tikki_agenda.git
    cd tikki_agenda
    python3 -m venv venv

Activate venv in macOS / Linux
    
    source venv/bin/activate

Install dependencies 

    pip install django

Migrate project

    python manage.py migrate

Run program
    
    python manage.py runserver
